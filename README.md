# Liar's Dice Roller

Mobile-friendly Liar's Dice roller by FringeliD, based on the small physical-table helper at https://kevan.org/games/liarsdice.

## Use

- Open `standalone.html` directly on a phone or computer for the single-file version.
- Open `index.html` for the multi-file version.
- Run `node serve.js` to serve it on the local network for phones on the same Wi-Fi.

## Version

- `v0.1` preserves the first multi-file release.
- `main` includes the standalone HTML version.
